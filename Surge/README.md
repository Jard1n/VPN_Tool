## 步骤



- ### SSH登录到VPS



- ### 获取权限



`sudo -i`



- ### 更新和升级系统



`apt update && apt upgrade -y`

如果没有安装过 Python3 的话要安装

- ### 安装 Python3 和 pip 包管理器

  运行以下命令安装 Python3 和 pip 包管理器。



  ```
  sudo apt install python3 python3-pip
  ```





- ####  安装依赖包

  ```
  pip3 install psutil
  ```
  如果在安装psutil中出现错误提示，

按照提示信息安装命令改为：

   ```
  pip3 install psutil --break-system-packages
  ```
   ```
  或者
  apt update
  apt install python3-psutil -y
  ```
   ```
  在后面运行 py 程序的时候，如果遇到缺少的依赖包，会有相应的提示：ModuleNotFoundError: No module named xxxx  这时候通过上面的 pip3 install xxxx 进行相应的依赖包安装即可解决



- ###  编写运行程序



`vim /root/servertraffic.py`



将以下内容粘贴进去并保存



```
import http.server
import json
import psutil
import time
import os
from datetime import datetime, timedelta
from urllib.parse import urlparse, parse_qs

STATS_FILE = '/root/traffic_stats.json'

def get_current_cycle_start(reset_day):
    """根据重置日期计算当前周期的起始日期"""
    now = datetime.now()
    try:
        # 如果今天的日期大于等于重置日期，说明本月已经重置过了
        if now.day >= reset_day:
            cycle_start = now.replace(day=reset_day, hour=0, minute=0, second=0, microsecond=0)
        else:
            # 说明现在还在上一个计费周期的末尾（跨月了）
            first_of_this_month = now.replace(day=1)
            last_month_end = first_of_this_month - timedelta(days=1)
            # 处理部分月份没有31号的情况，取最小值
            target_day = min(reset_day, last_month_end.day)
            cycle_start = last_month_end.replace(day=target_day, hour=0, minute=0, second=0, microsecond=0)
        return cycle_start.strftime("%Y-%m-%d")
    except Exception:
        return now.strftime("%Y-%m-01")

def get_traffic_stats(reset_day):
    net_io = psutil.net_io_counters()
    curr_sent = net_io.bytes_sent
    curr_recv = net_io.bytes_recv
    
    # 动态获取当前周期的唯一标识（例如 "2023-10-15"）
    cycle_id = get_current_cycle_start(reset_day)
    
    stats = {
        "cycle_id": cycle_id,
        "total_sent": 0, "total_recv": 0,
        "last_hw_sent": curr_sent, "last_hw_recv": curr_recv
    }

    if os.path.exists(STATS_FILE):
        try:
            with open(STATS_FILE, 'r') as f:
                saved_stats = json.load(f)
                # 只有周期 ID 一致（即还在同一个月度周期内）才读旧数据
                if saved_stats.get("cycle_id") == cycle_id:
                    stats = saved_stats
        except: pass

    # 计算差值（考虑重启情况）
    diff_sent = curr_sent - stats["last_hw_sent"]
    diff_recv = curr_recv - stats["last_hw_recv"]
    if diff_sent < 0: diff_sent = curr_sent
    if diff_recv < 0: diff_recv = curr_recv
    
    stats["total_sent"] += diff_sent
    stats["total_recv"] += diff_recv
    stats["last_hw_sent"] = curr_sent
    stats["last_hw_recv"] = curr_recv
    stats["cycle_id"] = cycle_id

    with open(STATS_FILE, 'w') as f:
        json.dump(stats, f)
    return stats

class SimpleHTTPRequestHandler(http.server.BaseHTTPRequestHandler):
    def do_GET(self):
        # 解析 URL 参数
        query_components = parse_qs(urlparse(self.path).query)
        # 获取重置日期，默认为 1 号
        reset_day = int(query_components.get("reset_day", [1])[0])
        
        traffic = get_traffic_stats(reset_day)
        cpu = psutil.cpu_percent(interval=None)
        mem = psutil.virtual_memory().percent
        uptime = int(time.time() - psutil.boot_time())
        
        data = {
            "cpu_usage": cpu,
            "mem_usage": mem,
            "bytes_sent": traffic["total_sent"],
            "bytes_recv": traffic["total_recv"],
            "bytes_total": traffic["total_sent"] + traffic["total_recv"],
            "uptime": uptime,
            "cycle_start": traffic["cycle_id"],
            "last_time": datetime.now().strftime("%H:%M:%S")
        }
        
        self.send_response(200)
        self.send_header('Content-type', 'application/json')
        self.send_header('Access-Control-Allow-Origin', '*')
        self.end_headers()
        self.wfile.write(json.dumps(data).encode())

print("Monitor Server running on 7122 with Dynamic Reset Support...")
httpd = http.server.HTTPServer(('0.0.0.0', 7122), SimpleHTTPRequestHandler)
httpd.serve_forever()
```

**说明：此代码是通过VPS的7122端口进行监控，请确保该端口的开放，如果你熟悉代码，也可以根据自己需要进行修改**



- ###  编写服务



`vim /etc/systemd/system/servertraffic.service`

将下面内容粘贴进去后保存



```
[Unit]
Description=Server Traffic Monitor

[Service]
Type=simple
WorkingDirectory=/root/
User=root
ExecStart=/usr/bin/python3 /root/servertraffic.py
Restart=always

[Install]
WantedBy=multi-user.target
```



- ###  运行



  进行完上述步骤后，执行下面指令运行

启动并激活这个服务：



`sudo systemctl start servertraffic.service`

`sudo systemctl enable servertraffic.service`



查看服务状态以确保它已经运行：



`sudo systemctl status servertraffic.service`



这样就可以在后台运行 /root/servertraffic.py 了。请按照自己实际情况修改服务单元文件中的相关配置，比如 User 和 WorkingDirectory。
